TTMS0800 - Web-palvelun hallinta (osaamistesti)

Jämbeck Marja, K8691, 7.4.2020

TEHTÄVÄ 1

Itsearvio: 6/6p VALMIS. Toimii.

Testin mukainen Ubuntu asennettu valmiista vaaditusta levykuvasta.

Asensin Apachen.
sudo apt install apache2
...
Toiminta testattu ja palvelin toimii virtuaalikoneen IP-osoitteessa.

Kopioitu Jyvaskyyla-sivu komennolla:
sudo wget --mirror -p --convert-links -P /var/www/html/jyvaskyyla netisto.fi/jyvaskyyla/

Tiedostot kopioituivat vahingossa netisto.fi -alakansioon, joten piti kopioida ne ylemmäksi hakemistossa komennolla:
sudo cp -a /var/www/html/jyvaskyyla/netisto.fi/jyvaskyyla/* /var/www/html/jyvaskyyla
Nyt sivusto on toiminnassa osoitteessa http://linuxin.ip.numero/jyvaskyyla/

TEHTÄVÄ 3

Itsearvio: 3/3p VALMIS. Toimii.

FTP asennettu.
sudo apt install vdftpd

Tarkistettu toiminta:
sudo systemctl is-active vsftpd
sudo systemctl status vsftpd

Muokataan vsftpd.conf missä sallitaan write_enable ja local_enable:
sudo nano /etc/vsftpd.conf

Testattu yhteys WinSCP:llä testi-käyttäjän tunnuksilla.
Yhteys toimii ja tiedostoja voi siirtää etätyöasemasta (isäntäjärjestelmästä) Linux-palvelimelle. Kuvassa kopioitu osaamistesti.html virtuaalikoneelle.
